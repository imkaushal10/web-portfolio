# Portfolio Website

A full-stack web application personnal portfolio showcasing projects, skills, and professional experience.

## Features

- Responsive design
- Project showcase
- Contact section
- Dark/light theme support

## Technologies

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js/Express
- Database: MongoDB

## Getting Started

```bash
git clone <repository-url>
cd portfolio-website
npm install
npm start
```
